# ch03 模擬下載資料夾

模擬一個亂七八糟的「下載」資料夾，讓讀者跟著 3.1 練習檔案分類。

## 使用方式

1. 把這些檔案的 `.txt` 副檔名去掉（例如 `catmeme.jpg.txt` → `catmeme.jpg`），或直接用這些 .txt 檔練習
2. 全部丟到一個資料夾裡
3. 跟著書裡 3.1 的指令，請 Claude Code 按副檔名分類
