# 測試資料說明

各章搭配使用的測試資料，讀者可以用這些資料跟著書中步驟實際操作。

## ch05 文字處理
- `ch05_meeting_transcript.txt` — 模擬 Whisper 產出的會議逐字稿（無標點、有填充詞）
- `ch05_interview_01_wang.md` ~ `ch05_interview_05_lin.md` — 五份採訪筆記（產品、設計、工程、行銷、業務）
- `ch05_recipe_sample.txt` — 從食譜網站複製貼上的原始內容

## ch06 文件處理
- `ch06_contract_sample.txt` — 模擬合約內容（含金額、到期日、自動續約條款、違約金、管轄法院）

## ch08 批次生產力
- `ch08_clients.csv` — 10 筆客戶名單（公司、聯絡人、Email、產品、報價金額）
