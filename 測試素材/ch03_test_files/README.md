# ch02 測試資料

讓讀者跟著第 2 章實際操作 .claudeignore 功能。

## 使用方式

1. 把這整個資料夾複製到桌面（或任何你想練習的地方）
2. 用 VS Code 打開這個資料夾
3. 跟 Claude Code 說：「讀一下這個資料夾裡所有的檔案，告訴我每個檔案的內容」
4. 觀察哪些檔案被讀到、哪些被 .claudeignore 擋住

## 預期結果

| 檔案 | 應該被讀到？ | 原因 |
|------|-------------|------|
| 一般文件.txt | ✓ 是 | 不在排除規則裡 |
| .env | ✗ 否 | 被 `.env` 規則排除 |
| 機密資料.txt | ✓ 是 | 檔名不含「密碼」或「password」，不在規則裡 |
| password_list.txt | ✗ 否 | 被 `*password*` 規則排除 |
| 報稅/110年度綜合所得稅.txt | ✗ 否 | 被 `報稅/` 規則排除 |

注意：「機密資料.txt」故意沒有被擋住，用來說明 .claudeignore 只認規則不認內容。如果你想擋住它，需要在 .claudeignore 裡加上 `*機密*`。
