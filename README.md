# 不會寫程式也能用的 AI 秘書：Claude Code 實戰指南

這本書教你用 Claude Code 當 AI 秘書，不需要會寫程式，不需要有資工背景。你只要會打字、會描述你想做什麼事，剩下的交給 AI。

## 測試素材

書中每章的實作練習都會用到測試素材，已經放在這個 repo 裡對應的資料夾中。下載方式：

點綠色的「Code」按鈕，選「Download ZIP」，解壓縮後把整個資料夾丟到你的 AI Space 裡面。

| 資料夾 | 對應章節 | 內容 |
|--------|----------|------|
| ch02_downloads/ | 第二章 | 模擬電腦下載資料夾（各種混亂的檔案） |
| ch02_images/ | 第二章 | 5 張 AI 產生的假照片（整理練習用） |
| ch03_test_files/ | 第三章 | 安全性測試用檔案 |

## 這本書適合我嗎？

如果你符合以下任何一項，這本書就是為你寫的：

- 每天花大量時間做重複性的文書工作
- 聽過 AI 很厲害但不知道怎麼用在自己的工作上
- 試過 ChatGPT 覺得不夠用，想要 AI 能直接幫你操作電腦
- 不會寫程式，但想讓 AI 幫你寫腳本、自動化流程
