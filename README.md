# 不會寫程式也能用的 AI 秘書：Claude Code 實戰指南

這本書教你用 Claude Code 當 AI 秘書，不需要會寫程式，不需要有資工背景。你只要會打字、會描述你想做什麼事，剩下的交給 AI。

## 免費試閱（第 1-5 章）

| 章節 | 標題 | 內容 |
|------|------|------|
| [第 1 章](chapter_01.md) | 自己當老闆：請個 AI 秘書 | 安裝 Claude Desktop、建立工作資料夾、第一次對話 |
| [第 2 章](chapter_02.md) | 秘書上工了 | 下載資料夾整理、匯率追蹤腳本、照片廢片辨識 |
| [第 3 章](chapter_03.md) | 門禁卡管理：安全性與權限 | 黑名單/白名單權限設定、.claudeignore、Hook 通知 |
| [第 4 章](chapter_04.md) | 跟秘書高效溝通 | 省 token 四招、腳本重複利用、操作技巧 |
| [第 5 章](chapter_05.md) | 讓秘書記住你的規則 | CLAUDE.md 員工手冊、Skill 快捷指令、Memory 記憶 |

## 這本書適合我嗎？

如果你符合以下任何一項，這本書就是為你寫的：

- 每天花大量時間做重複性的文書工作
- 聽過 AI 很厲害但不知道怎麼用在自己的工作上
- 試過 ChatGPT 覺得不夠用，想要 AI 能直接幫你操作電腦
- 不會寫程式，但想讓 AI 幫你寫腳本、自動化流程
